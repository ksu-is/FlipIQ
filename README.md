# FlipIQ 🃏
A command-line flashcard study application built in Python.
Students can load any subject from a .txt file and quiz themselves instantly.

## How to Run
1. Make sure Python is installed
2. Create a flashcard .txt file using the format: `Term|Definition`
3. Run the program: `python flipiq.py`

## Features
- Load flashcards from a .txt file
- Quiz mode: see the definition, type the term
- Score report at the end of each session
- Missed cards saved for follow-up study

## Built With
- Python 3
- Core concepts: file I/O, functions, loops, lists, string methods
